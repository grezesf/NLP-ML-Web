maximize assonance in a string(interview question)

up vote 1 down vote favorite

Question: Assonance is the reiteration of the same vowel sound at the beginning of several consecutive words. Assume that each vowel(A, E, I, O, and U) has only one sound. Prompt the user for an alpha-only string with spaces as delimiters between words. Your program will arrange the words to maximize assonance, and print the new string. After the first word in the string that begins with a given vowel, and all other words in the string that begin with the same vowel must immediately follow that vowel, in the order they appear in the original string. All non-assonized words should remain in the same order in which they occurred in the original string.

Example:

Input:

“assonance reiteration is the of the same vowel sound at the beginning of several consecutive words”

Output:

"assonance at reiteration is the of of the same vowel sound the beginning several consecutive words"

My idea:

(1) define vector<vector<string> > r;

(2) traversal the word in the string one by one.

(3) if the word begins with 'a', 'e', 'i', 'o' and 'u', insert it to the vector of string which has the same vowel.

(4) if the word doesn't, just insert the word to the back of r

The following is my code:

void split_by_whitespace(string& s, vector<string>& result)
{
    stringstream ss(s);
    string cur;

    while(ss >> cur)
        result.push_back(cur);
}

void maxi_assonance(vector<string>& x)
{
    vector<string>::iterator it = x.begin();
    vector<vector<string> > r;
    map<char, int> key_ind; 

    for(; it!=x.end(); ++it)
    {
        string cur = *it;
        if(cur[0]=='a' || cur[0]=='e' ||
           cur[0]=='i' || cur[0]=='o' ||
           cur[0]=='u')
        {   
            map<char, int>::iterator it_map = key_ind.find(cur[0]);
            vector<string> tmp;
            if(it_map==key_ind.end())
            {
                key_ind.insert(pair<char, int>(cur[0], r.size()));
                tmp.push_back(cur);
                r.push_back(tmp); 
            }
            else
            {
                r[key_ind[cur[0]]].push_back(cur);
            }
        }
        else
        {
            vector<string> tmp;
            tmp.push_back(cur);
            r.push_back(tmp);
        }   
    }

    vector<vector<string> >::iterator it1 = r.begin();
    for(; it1!=r.end(); ++it1)
    {
        vector<string> cur = *it1;
        vector<string>::iterator it2 = cur.begin();
        for(; it2!=cur.end(); ++it2)
        {
            cout << *it2 << " ";
        }
    }
}

split_by_whitespace(s, splitted); 
maxi_assonance(splitted);
