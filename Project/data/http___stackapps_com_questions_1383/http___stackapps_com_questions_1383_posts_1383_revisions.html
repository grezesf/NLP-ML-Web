Is it possible to get this explained/sorted without dissecting any case individually?

Update:

George Edisons comments to my question appear to explain most issues raised here (if not all), so please disregard this question for a moment, as unfortunately I have to leave right now for a road trip - I'll update my question on Sunday/Monday as appropriate.

While one issue of reputation inflation via multiple badge assignment has been resolved recently, the root cause has apparently not been tackled yet, as easily seen in the just officially announced new per-site reputation leagues:

  • Sorting by August
    • yields 1 user being rewarded the API Beta badge for 2k reputation despite being a member as of Aug 12 only
  • Sorting by July
    • yields 3 users being rewarded the API Beta badge for 2k reputation, where one is a member as of May 21 only - don't know the exact cutoff date, but the Public Beta announcement has been on May 20
  • Sorting by June
    • yields 3 users being rewarded the API Evangelist badge for 4k reputation, where some don't have any posts before May 21 and some don't have any posts at all - see my related question What constitutes 'active participation' in the private API beta?
    • yields 5 users being rewarded the API Beta badge for 2k reputation, where some are a member as of May 21 or even June only - again, don't know the exact cutoff date, but the Public Beta announcement has been on May 20

There might be several issues involved which might confuse the whole picture, but something seems wrong here or confusing at best:

  • Maybe the cutoff date is a 24 hour window bordering into May 21, which might explain some API Beta badges to be correct, but some of these May 21 cases have been rewarded a API Evangelist badge without any post still.
  • Maybe the whole registration/membership has been reset after the private beta and the badges are triggered by the next visit only, but again, this doesn't explain the API Evangelist badges and would qualify as a confusing bug in the 'member for' display at least.
  • Looking at the API Evangelist badges in isolation yields users with posts after May 20 only and no posts or no activity at all even.

Is it possible to get this shadow economy explained/sorted without dissecting any case individually? ;)
