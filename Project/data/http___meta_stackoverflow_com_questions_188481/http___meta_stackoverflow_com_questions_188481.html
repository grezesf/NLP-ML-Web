Second bounty should be doubled the amount only if there is an awarded bounty in the question

up vote 16 down vote favorite
1

I've offered a +500 bounty in this question and it has not been given to any answer. However, I would like to offer another +50 bounty to award one answer, but the system only allows me to award +500.

It would be great if, in the case where the bounty is not awarded, the next one does not have to be doubled.

EDIT: it happened to this question too... if I try to start a bounty here the minimum is 100, even tough nothing has been given...

This limitation was put into place because people would use the bounty to attract attention to earn up votes worth more reputation than the bounty they offered, so it probably doesn't need to be enforced in cases where the bounty poster can't earn more reputation (i.e. they didn't ask the question or answer).
