According to majority of scholars, If the male organ enters woman even once, then Ghusl is obligatory on both. Ejaculation is not the condition for Ghusl, entering into woman is.

Also, If ejaculation happens without penetration, even then ghusl is fard. So in both cases ghusl is fard.

So you must do ghusl before praying, otherwise prayer will not be accepted

If there was no actual penetrative intercourse, and no ejaculation / orgasm, then ghusl (bath) is not required. If only one of the couple experienced orgasm / ejaculation, then that individual is required to perform ghusl. If there was penetrative intercourse, then both man and woman must perform ghusl, even if there was no ejaculation nor orgasm.

source: http://spa.qibla.com/issue_view.asp?HD=12&ID=12832&CATE=10
