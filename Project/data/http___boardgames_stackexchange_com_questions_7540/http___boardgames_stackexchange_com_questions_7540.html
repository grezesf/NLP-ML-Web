Where can I find a printer friendly version of the settlers of catan rulebook?

up vote 7 down vote favorite

My copy of Settlers of Catan has been missing the rule book for a while. The Catan official site has the rule book available for download, but since it contains a lot of block backgrounds, it's not really printer friendly.

Is there anywhere I can download a printer friendly version of the base game full rules?

This appears to be a version of the original rules. The only rules differences (from 4th Edition) would be:

  • That now you can move the Robber to the desert.
  • Trading and Building are not distinct phases.
  • Harbors are now part of the island border.
