The biggest awkwardness is that the original \SpotColor command expects the color tint as an argument, while TikZ expects a singe color name as defined by xcolor.

To make this work together, I have patched \AddSpotColor to define a color with tint 100% under the name of the spot color, and there is a new command \definespotcolortint to define a spot color tint under a unique name (see example).

There are certainly ways to make the interaction more smooth, for instance to enable using a spot color name directly as a TikZ option, or enabling to directly switch to a spot color with \color.

The biggest awkwardness is that the original \SpotColor command expects the color tint as an argument, while TikZ expects a singe color name as defined by xcolor.

To make this work together, I have patched \AddSpotColor to define a color with tint 100% under the name of the spot color, and there is a new command \definespotcolortint to define a spot color tint under a unique name (see example).

There are certainly ways to make the interaction more smooth, for instance to enable using a spot color name directly as a TikZ option, or enabling to directly switch to a spot color with \color.

User Pygmalion found an error (\colorlet should not be global) and asked about how to combine black and spot colors.

  1. The error should be fixed above.
  2. To combine black with spot colors, just use colors (like black) defined in the gray color model.

Using your test file, with fix to the error and reference to black and a new color grey:

\documentclass[a4paper]{article}

\usepackage[margin=15mm]{geometry}
\usepackage{spotcolor}
\usepackage{tikz}
\usepackage{etoolbox}

\makeatletter

\newcommand\@@spotcolor{spot}

\apptocmd\AddSpotColor
{%
  \expandafter\xdef\csname\string\color @#2\endcsname
  {{}{}{}{spot}{#2,1.0}}%
  \global\expandafter\let
  \csname @@colortype@#2\endcsname\@@spotcolor
}{}{}%

\def\pgfsys@color@spot@stroke#1#2{\pgfsysprotocol@literal{/#1 CS}\pgfsysprotocol@literal{#2 SC}}
\def\pgfsys@color@spot@fill#1#2{\pgfsysprotocol@literal{/#1 cs}\pgfsysprotocol@literal{#2 sc}}
\def\pgfsys@color@spot#1#2{\pgfsys@color@spot@stroke{#1}{#2}\pgfsys@color@spot@fill{#1}{#2}}

\let\o@XC@col@rlet\XC@col@rlet
\def\XC@col@rlet[#1]#2[#3]#4%
{%
  \ifcsname @@colortype@#4\endcsname
    \expandafter\let
    \csname @@colortype@#2\endcsname\@@spotcolor
    \expandafter\let
    \csname\string\color @#2\expandafter\endcsname
    \csname\string\color @#4\endcsname
   \else
    \expandafter\let
    \csname @@colortype@#2\endcsname\@undefined
    \o@XC@col@rlet[#1]{#2}[#3]{#4}%
  \fi
}%

\newcommand\definespotcolortint[3]
{%
  \expandafter\xdef\csname\string\color @#1\endcsname
  {{}{}{}{spot}{#2,#3}}%
  \global\expandafter\let
  \csname @@colortype@#1\endcsname\@@spotcolor
}

\makeatother

\definespotcolortint{PANTONE3035PCA}{PANTONE3035PC}{.5}
