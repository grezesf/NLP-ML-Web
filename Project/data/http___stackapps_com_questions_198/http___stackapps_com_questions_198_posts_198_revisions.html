Python 3.x is also supported - a simple run of the file through '2to3' like this:

py-stackexchange $ 2to3 ./*.py > 2to3.diff
py-stackexchange $ patch -p0 < 2to3.diff

Will do the trick. You might want to keep the diff around, so you can reverse the operation with the -R flag if necessary.
