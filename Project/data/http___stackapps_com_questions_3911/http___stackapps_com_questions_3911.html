StackWatcher - Google Chrome Extension

Screenshot

About

There are a lot of very smart people on StackOverflow and most of them are smarter then me. I think even if you're not going to respond to any questions it's worth just reading other developers' questions and answers.

I made this extension about a year ago but I wasn't able to finish some important features and bug fixes until now. Shame on me!

Right now it supports only StackOverflow but I want to include all Stack Exchange sites.

License

It's free.

Download

You can install it from Chrome Web Store.

Platform

Google Chrome

Contact

I would love to hear any comments, suggestions or feature requests.
Just leave a comment here or reach me on Google+ or use contact form on my blog.

Code

This is pure HTML5, CSS3 and tons of JavaScript.
I used jQuery, Backbone.js, SyntaxHighlighter, ... just to name the most important.
