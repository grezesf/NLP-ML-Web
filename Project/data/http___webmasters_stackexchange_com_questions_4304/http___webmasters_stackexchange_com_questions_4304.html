There could be several reasons for this:

  1. Your visitors are using adblock software so the ads are never shown

  2. Adsense can't find any ads that fit your site and is showing public service advertising

  3. Your Adsense code is not being loaded either because it is incorrectly applied to your website or the page stops loading before it gets to that code

  4. Users have JavaScript turned off

You can contact Google in their Adsense support forums. Maybe someone there can help you identify the problem.
