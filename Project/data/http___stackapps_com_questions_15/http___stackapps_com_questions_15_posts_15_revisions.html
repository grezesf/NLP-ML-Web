StackTop

Stacktop is my Mac client for accessing most of the Stack Exchange sites.

Here are some screenshots of 3.0. Note the colors in the first screenshot change depending on which site you are "visiting":

This application will at first be much like a twitter client, then eventually I hope to make it more like an auto-search app where you input keywords then every 10 minutes or so it'll search SO and tell you if there are any questions that haven't been answered that are about whatever your tags are. Hopefully this will allow SO users to gain more rep and kick Jon Skeet out of #1 (Love you Jon!!!)

This will be released for free.

You can download it here here (Updated link. The application currently does not work, I will release an update with a version that actually works shortly :( !!!)

This will be for Mac ONLY

This was written entirely in Objective-C with the help of my Library, MSOEngine

Version 2.9 has been released.

  • Bug fixes
  • UI enhancements
  • Reworked answer window into a drawer. Makes the overall UX and UI a little cleaner. I'll post up screenshots of the new UI a little later.

Known Bugs:

  • UI Bug in Question Display window, not really a big deal, I'll fix it in 3.0, the official public build.

VERSION 3.0!

StackTop is now out of beta!!!

Contains all the features listed in the 3.0 section, MINUS comments. I figured comments wasn't a huge deal, so I left them out. Plus, I couldn't figure out a good way to integrate them

Version 3.0.1 released

-Bug Fixes

Contact

You can contact me at matt (at) mattsapps (dot) com

Public Release (3.0) Features

I'm very hard at work on StackTop 3.0, so far, here are the new features:

  • Tags (finally)
  • Auto Search, replacing that placeholder in the preferences :)
  • Comments (maybe...)
  • Stack Apps Support (Again, finally)
  • Reworking some behind the scenes things so EVERY question loads (right now it's at about 99%)

Notes

I will NOT be supporting other SE sites UNTIL they are out of beta. If you've ever used StackTop, then I'm sure you can imagine why. It really is a pretty big pain to get those colors in the table view (mainly the highlight colors), so I'm not doing it until the site is part of the SE network. Also, I will not be including the new metas for super user and server fault until there is a new site in the SE network.
