A question about Answer Time

up vote 1 down vote favorite

In my opinion, first given best answer should take up vote. In an hour, we can see which one is given first like (10 min ago, 15 min ago, etc.) But after an hour, these times looks like 1h ago on every answer given in 1h ago. So, Im confusing which one is given first. Some users can make copy/paste or gives similar answers. Is it possible to add minute detail in answers? For example, "answered 1h 22m ago", "answered 1h 36m ago". So I can see that which answer is best and given early :)

Is it possible on Stack Overflow?
