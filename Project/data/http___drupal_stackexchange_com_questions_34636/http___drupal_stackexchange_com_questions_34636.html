How do I print a taxonomy term in node.tpl.php? (D7)

up vote 1 down vote favorite

I have a custom content type that I can select a vocab term for when editing the page (category more or less). How do I print this term to my page via node.tpl.php?

I've tried several different ways, but to no avail so far. Any ideas?

This works for me:

<?php print render($content['field_tags']); ?>
