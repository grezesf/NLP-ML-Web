If you do not want the solution to be sandboxed, mark the project in Visual Studio and from the Properties Windows set "Sandboxed Solution" to False.

If the solution needs to be Sandboxed you can not use some kind of artifacts, as your error message states. One of these is Visual Webparts!

Here is more about what you can not do in Sandboxed solutions (for 2010, but also valid for 2013):

http://jaliyaudagedara.blogspot.se/2012/11/sharepoint-sandboxed-solutions-vs-farm.html http://msdn.microsoft.com/en-us/library/ee231562%28v=vs.100%29.aspx
