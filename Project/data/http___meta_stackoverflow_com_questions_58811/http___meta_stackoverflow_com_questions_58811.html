This is a great idea! Finally we have an incentive to not give up on people and encouragement to generate great questions from confused seeds.

I seriously love the idea.

I don't necessarily think the implementation will work.

  1. This may encourage people to completely change the question, rather than just reform it.

  2. There are some questions which actually need to be closed. If they are offensive or abusive, we would want the user to be punished for the behaviour, not rewarded with upvotes.

  3. This could also be gamed by piggy-backing on someone else's edit.

  4. Because the OP is still getting upvotes, they haven't really learned any sort of lesson about writing good posts.

I think implementation problems like the ones in devinb's answer were the reason this failed the last time it came up, but there wasn't much discussion about it. I really would like to see this happen, because people have a bad habit of seeing a poorly worded but perfectly valid question and moving their mouse straight past "edit" to "close", which I would very much like to discourage. It becomes a race, where the editors try to fix up the question before the closers get 5 votes in, because once the question is fixed the close votes will stop, but if it closes first it's almost impossible to reopen
