Generally, just drag the application from it's folder to the trash. You may also use spotlight to locate files with the same name. Most all support files for an application are small and can be ignored.

If the program has a dedicated uninstaller, use it. Otherwise simply trashing the application is enough. If you want to be extra thorough, you can trawl /Library and ~/Library for left overs.
