FIX/FAST packet analyzer

up vote 3 down vote favorite

I have a network dump of market data in the FIX/FAST format (transported over UDP). Are there standard pieces of software (similar to Wireshark) that will parse and allow for inspection of the key-value pairs of the payload?

Try the FAST Antenna Codec for Windows. You need a little wrapper app that reads the UDP data from your dump and feeds the payload (i.e. sans IP/UDP headers) to the codec.
