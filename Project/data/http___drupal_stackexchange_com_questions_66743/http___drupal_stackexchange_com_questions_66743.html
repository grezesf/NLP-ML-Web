If you need to import, I would suggest the migrate module But it depends on how your content is put together. All though using Migrate module covers you in almost every situation.

To see examples you want to look at

Beer.inc (Basic file migration, you can use those code snippets/examples it provides for your migration)

Wine.inc (Advanced Migrations, if you need something more advanced this would be the file too look at)

Its all in the migrate module folder

Besides that

Migrate module has everything you need

Multiple files in a node

This was also a great help

Big file imports with the Migrate module

If your new to the migrate module I would you suggest reading the documentation first

Migrate Module Documentation

Tutorial on Drupal Migration (the concept is the same)

If your not OOP experienced you might find it a bit difficult.
