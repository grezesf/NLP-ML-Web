Lucene search not finding an address in address feature class

up vote 0 down vote favorite

I have a few addresses in my address feature class that are not being found in my web application which uses the Lucene indexing tool. The Lucene index is updated on a nightly basis so I don't think it's a matter of these records not being found in the Lucene index.

Most of the times the Lucene search works properly so... 1) I start typing in an address in my address search 2) pick the address from the pick list that gets created 3) click enter and am zoomed to that address in my web app

With a few addresses the behavior isn't correct, or isn't what is desired... 1) I start typing in an address in my address search 2) pick the address from the pick list that gets created 3) a box pops up saying the address I searched for a a partial match 4) I need to click the address again before I am zoomed to that location in the map

I welcome feedback from people that use Lucene indexing in a web map application. Thanks & regards.
