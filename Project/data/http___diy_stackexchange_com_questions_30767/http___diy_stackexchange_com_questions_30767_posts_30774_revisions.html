You'll create a mess if you don't predrill the particle board because the screws will at least break the backside and never draw it down ... especially if it is not high density PB (heads will pull through) ... and especially going into something as thin as 5/8" subfloor grade plywood.

When predrilling, you'll want a depth stop to keep from drilling into the 5/8" because if you over drill by 1/8", the screw will have maybe 1/2" to grip. A small chunk of wood can be used as a positive depth stop. It'd be best if you layout the joist locations and try to screw into those.

If you are using adhesive from a caulking gun, you'll end up with a wavy floor if you were to put a straight edge to it afterwards.

Regardless, I can't see the job coming out well using 3/8" particle board. Sorry for the pessimism. Is 3/4" not an option?
