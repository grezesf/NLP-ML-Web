I fetched some rows on my custom admin page (using hook for adding admin menus). I wanted to add a search box on that page to display only those rows with the 'name' column containing searched string in that column.

ID     |    Name
_________________
1      |    abc
2      |    abc
3      |    abz
4      |    xyz
_________________

I was trying with

$keyword = $_POST['data'];
$poi_sql="SELECT * FROM Details WHERE Name LIKE %".$keyword;

I got this::

$keyword = $_POST['data'];
$poi_sql="SELECT Name FROM Details WHERE Name LIKE '%". $keyword ."%'";

Silly mistakes. P)
