You can also default this to, for example, sites/all/modules/contrib, in your .drushrc file: $command_specific['dl'] = array('destination' => 'sites/all/modules/contrib'); Also note, you won't be able to exclude the 'modules' part of the path, as Drupal uses this to scan for modules. –  jhedstrom Mar 24 '11 at 1:46

As requested, posting my comment as a proper answer:

You can default this to, for example, sites/all/modules/contrib, in your .drush/drushrc.php* file:

$command_specific['dl'] = array('destination' => 'sites/all/modules/contrib');

Also note, you won't be able to exclude the 'modules' part of the path, as Drupal uses this to scan for modules.

*There are 6 places this file can live.
