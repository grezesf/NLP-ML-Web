I'd suggest you do the following:

  1. Start with a list of things the feature is supposed to do. You should already have this from when you created the feature. If you have a function specification, use that to create the list.
  2. For each item on the list write some test cases to test the item.
  3. Edit your question and add the list of tests you created. Ask if your test cases are good or not.
