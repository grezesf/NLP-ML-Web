Java POI SS usermodel Conversion CellStyle

up vote 0 down vote favorite

Im converting form HSSF to the SS model.

There is a section in our code where we check the cellstyle and so some comparisons.

In the new ss model there is not a method that gets the style name like there is in the HSSF model.

We are doing something like this:

   private static CellStyle findStyleByName(Workbook wb, String styleName)
{
    for ( short i = 0; i < wb.getNumCellStyles(); i++ )
    {
        CellStyle style = wb.getCellStyleAt(i);

        if ( styleName.equals(style.getUserStyleName()) )
        {
            return style;
        }
    }

    return null;
}

This fails because style.getUserStyleName() is not valid with CellStyle it only is with HSSFCellStyle

Is there an alternative method or way I can do this using the ss usermodel?
