Evaluating $\int\frac{\sin x}{\sin (x-a)\cdot \sin (x-b)}\,\mathrm dx$

up vote 4 down vote favorite
1

$$\int\frac{\sin x}{\sin (x-a)\cdot \sin (x-b)}\,\mathrm dx$$

My Try::

\begin{align} &\displaystyle \frac{1}{\sin (b-a)}\int\frac{\sin \{(x-a)-(x-b)\}}{\sin (x-a)\cdot \sin (x-b)}\cdot\sin x\,\mathrm dx\\ =& \frac{1}{\sin (b-a)}\int \left\{\cot (x-b)-\cot (x-a)\right\}\cdot \sin x \,\mathrm dx \end{align}

Now how can I proceed after that?

The Maple command $$simplify(int(sin(x)/(sin(x-a)*sin(x-b)), x)) $$ outputs $$- \left( \cos \left( a \right) \sin \left( b \right) -\sin \left( a \right) \cos \left( b \right) \right) ^{-1}\times $$ $$\left(2\,\sin \left( b \right) {\it arctanh} \left( {\frac {\cos \left( b \right) \sin \left( x \right) -\sin \left( b \right) \cos \left( x \right) +\sin \left( b \right) }{\sin \left( x \right) }} \right) -2 \,\sin \left( a \right) {\it arctanh} \left( {\frac {\cos \left( a \right) \sin \left( x \right) -\sin \left( a \right) \cos \left( x \right) +\sin \left( a \right) }{\sin \left( x \right) }} \right)\right) $$
