What kind of dash is used in the combination of artist and song title?

up vote 2 down vote favorite

Which one is correct?

  • Def Leppard ‒ Photograph (figure dash, U+2012),
  • Def Leppard – Photograph (n-dash, U+2013),
  • Def Leppard — Photograph (m-dash, U+2014) or
  • Def Leppard ― Photograph (horizontal bar, U+2015).
