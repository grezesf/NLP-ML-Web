I have this problem occasionally when I'm connected to a very slow VPN (and in my case I need to route traffic over that network to get VPN-specific DNS to work correctly). I suspect you experienced some type of network hiccup; this is not a rendering problem, it is that your browser did not (completely) load the CSS to apply proper styling. I just gave Aurora a shot and all of the sites I frequent render just fine.
