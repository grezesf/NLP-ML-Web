Most keyboards come with suggestions for common words, so a spell check function is rendered moot. Once a user types a couple of letters, the most common words containing those letters usually show up in a bar above the on screen keyboard.

Most keyboards like Smart Keyboard / Better keyboard also have options that allow for word suggestion to show up if a user opts for the hardware keyboard instead of the "soft" on-screen keyboard.
