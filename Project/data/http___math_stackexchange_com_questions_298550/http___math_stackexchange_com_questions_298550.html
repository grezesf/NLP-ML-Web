find a counterexample of this statement of expression about the limit of composition function

up vote 0 down vote favorite

Suppose that $U$ is open in $\mathbb{R}^{m}$, that $L\in U$ and that $h:U\setminus \left \{ L \right \}\rightarrow \mathbb{R}^{p}$ for some $p\in N$. If $L=\lim_{x\rightarrow a}g(x)$ and $M=\lim_{y\rightarrow L}h(y)$. Then $\lim_{x\rightarrow a}(h\circ g)(x)=M$.

(Someone told me that this statement is false and should replace "$U\setminus \left \{ L \right \}$" with "$U$", and “$M=\lim_{y\rightarrow L}h(y)$. Then” with “$h$ is continuous at $L$, then”.)

So can you give me a counterexample of the original statement? Thanks.

A simple example is the following: Put $$g(x):\equiv1\ ,\qquad h(y):=\cases{2\quad&$(y=1)$\cr 3&$(y\ne1)$\cr}\quad.$$ Then $h\bigl(g(x)\bigr)\equiv2$ and therefore $$\alpha:=\lim_{x\to0}h\bigl(g(x)\bigr)=2\ .$$ On the other hand the "inner limit" $\lim_{x\to0} g(x)=1$, and being careless one therefore would write $$\alpha=\lim_{x\to0}h\bigl(g(x)\bigr)=\lim_{y\to1} h(y)=3\ .$$

Take $g(x)=L$ (constant), then $h(g(x))$ is not defined. If you suppose that $h$ is defined also in $L$, if it is not continuous you find: $$ \lim_{x\to a} h(g(x)) = \lim_{x\to a} h(L) = h(L) \neq \lim_{y\to L} h(y) $$
