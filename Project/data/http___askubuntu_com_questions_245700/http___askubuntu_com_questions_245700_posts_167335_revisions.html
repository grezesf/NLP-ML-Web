Step 1: Remove Graphics Card from your machine.
Step 2: Install Ubuntu (You shouldn't get any errors when running generic driver.)
Step 3: After Ubuntu is installed then put graphics card back into machine. With tower still off. Then swap your VGA/HDMI/DVI cables to the graphics card.

Turn on computer and select the recovery option from the GRUB menu then boot normally.

Step 4: Go to NviDia website and grab the linux driver it will come in .run format
Step 5: Install Driver using the following steps.
    Step 5a: Move the driver to the desktop and rename it something simple and easy to remember.
    Step 5b: CTRL+ALT+F1
    Step 5c: Run command "sudo /etc/init.d/lightdm stop" (Without Quotes on all listed.)
    Step 5d: Run command "cd ~/Desktop DRIVERNAME.run" (What you renamed it.)
    Step 5e: Run command "chmod +x DRIVERNAME.run"
    Step 5f: Run command "sudo ./DRIVERNAME.run"
    Step 5g: Follow instructions and continue
        **TIP** If you still get hung with "It appears you are running "X" server then change the command on Step 5c to as follows: "sudo /etc/init.d/gdm stop" (Without Quotes)
Step 6: CTRL+ALT+F7
Step 7: CTRL+ALT+T (Only use steps 7 & 8 if you can't access the restart button or see your screen.)
Step 8: Shutdown -h (OR sudo shutdown 0 -h if the other fails.)
Step 9: Boot computer and enjoy :) I suggest printing these instructions.
