The package caption provides different styles via option. To set one of the predefined styles use format=.

Here the an example with format=plain

\documentclass{scrartcl}
\usepackage[bf,format=plain,]\usepackage[bf]{caption} 
 \usepackage\def\exampletext{lipsumLorem ipsum dolor sit amet, consectetuer adipiscing elit. Ut purus elit,
vestibulum ut, placerat ac, adipiscing vitae, felis. Curabitur dictum gravida
mauris. Nam arcu libero, nonummy eget, consectetuer id, vulputate a,
magna.
}
\begin{document}
Text\begin{figure}[!ht]
\centering
\rule{7cm}{1cm}
\caption{\exampletext}
\end{figure}

\begin{figure}[!ht]
\captionsetup{format=plain}
\centering
\rule{7cm}{1cm}
\caption{\protect\lipsum[1]\exampletext}
\end{figure}

\begin{figure}[!ht]
\captionsetup{format=hang}
\centering
\rule{7cm}{1cm}
\caption{\exampletext}
\end{figure}
\end{document}

The commands \protect\lipsum[1] are used to create filling text.
