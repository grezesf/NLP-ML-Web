ATI Radeon Mobility 4200 - Display Manager Crashes When USB Devices Are Removed [closed]

up vote 1 down vote favorite

I installed Kubuntu 12.10 on my Toshiba Satellite with a USB device as a dual boot system. I have an ATI Mobility 4200 series graphics card, and I followed the instructions to install the drivers for ATI cards, however I receive a low resolution, fuzzy display after installing the ATI drivers.

I wouldn't use proprietary software if I didn't have to, but with the kernel provided driver I have problems with my display manager. I use a USB wireless adapter with the laptop and whenever I bump the adapter when I'm setting down the computer or plugging in the charger, the display manager crashes and I have to restart KDM from the tty screen. I also tested this with a previous install of Kubuntu and a KDE Mint installation, all of which returned similar results.

Is there any way to fix the display manager crashing whenever a USB device is removed, or bumped?
